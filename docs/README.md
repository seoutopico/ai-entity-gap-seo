# Documentación — Entity Gap (SEO/GEO con IA)

Sistema que coge tu archivo de contenido, detecta **sobre qué temas hay demanda que
no cubres** y te devuelve un **backlog editorial priorizado** — repetible, auditable
y manejable por conversación bajo Claude Code.

## Por dónde empezar

**¿No eres técnico?** Léelo en este orden:
1. [Qué es y para qué sirve](que-es.md) — el problema y la solución, en cristiano.
2. [Glosario](glosario.md) — qué significa cada palabra rara (salience, gap, pilar…).
3. [Cómo se usa](uso.md) — paso a paso, hablando con la IA.

**¿Eres técnico?** Ve directo a:
1. [El método](el-metodo.md) — las 7 fases, qué entra y sale en cada una.
2. [Arquitectura](arquitectura.md) — scripts / skills / agentes / arnés, config vs brief.
3. [Cómo se mide](como-se-mide.md) — todas las métricas y cómo interpretarlas.

## La idea en una frase

> Dato **cuantitativo → script** (determinista). Dato **cualitativo → IA** como
> clasificador, no como oráculo. Nada se da por hecho sin evidencia.
